# Журнал змін (Changelog)
Усі помітні зміни в цьому проєкті будуть документуватися в цьому файлі.

## [0.1.0] - 2026-03-25
### Додано
- Скрипт `load_data.py` для коректного розпізнавання та завантаження Excel-файлів (`.xlsx`), замаскованих під CSV, з порталу відкритих даних.
- Скрипт `visualize.py` для побудови стовпчикових діаграм розподілу вільних приміщень за районами Києва.
- Базовий скрипт аналізу якості даних (`data_quality.py`).
- Налаштовано базову інфраструктуру проєкту та віртуальне середовище.
