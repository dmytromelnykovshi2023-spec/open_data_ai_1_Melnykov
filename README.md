**Назва:** Аналіз фонду комунальної нерухомості та просторова кластеризація вільних житлових приміщень за допомогою методів машинного навчання.

**Мета:** Дослідження характеристик незаселеного житлового фонду (геолокація, площа, стан будівлі), виявлення просторових і статистичних закономірностей та розробка предиктивної моделі для автоматичної оцінки інвестиційної привабливості об'єктів (наприклад, прогнозування оптимальної вартості оренди, ймовірності приватизації або пріоритетності проведення ремонтних робіт).

**Джерело відкритих даних:** Реєстр вільних житлових приміщень (квартир) комунальної власності, що оприлюднюється органами місцевого самоврядування (data.gov.ua) — https://data.gov.ua/dataset/21e5c671-7dae-4261-a366-60f5d5f4ec7b
